# Bank Transaction Visualizer

This project is a web-app that takes in comma-separated value (csv) files containing bank transactions, and automatically creates graphs to visualize them.
Your transactions are kept local to your computer, and are not uploaded to any servers.

## Using:

https://tombez.github.io/bank-transaction-vis

## Features:

* Automatic transaction categorization
* Category hierarchical pie graph
* Account activity over time graph
* Account balance over time graph
* Spending and income flow graph (sankey chart)
* Automatic transaction validation utilizing balance information
* Transaction filtering
* Net worth over time graph
